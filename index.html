Network Sockets

The three main types:

(Use arrows to navigate)

1. Datagram
2. Stream
3. Raw

Datagram Sockets:

  • Connectionless
  • Use UDP

Stream Sockets

  • Connection-oriented
  • Use TCP (or SCTP, DCCP)

Raw Sockets

Allow direct sending and receiving of IP packets without any protocol-specific transport layer formatting.1
